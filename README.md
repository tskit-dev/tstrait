**tstrait** is a quantitative trait simulator of [tskit](https://tskit.dev) tree sequences. It supports simulation of quantitative traits, sampling genetic effect sizes from various distributions, modelling pleiotropic phenotypes and environmental noise, and computing perâ€‘individual genetic values given the effect sizes of causal mutations.

Documentation: https://tskit.dev/tstrait/docs/stable/ (latest: https://tskit.dev/tstrait/docs/latest/)
Learn more about tree sequences: https://tskit.dev/learn/

## Installation

```bash
python -m pip install tstrait
# or
conda install -c conda-forge tstrait
```
